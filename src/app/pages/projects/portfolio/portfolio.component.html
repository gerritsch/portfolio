Description

One of my occasional frontend projects is this personal website which functions as a portfolio of my personal projects. The goal of this project was to build a simple minimalistic website with the new version 9 of Angular and to learn more about Netlify and their hosting capabilites for Angular Applications.



Used Technologies

  • Angular Version 9
  • Netlify for Hosting
  • PrimeNG for UI Components
